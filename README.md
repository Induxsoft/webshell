# Webshell (Entorno Web de V12). 
El paquete Webshell (Entorno Web de V12) es el módulo que provee las bibliotecas y componentes de la interfaz de usuario integrada, además de herramientas para la administración de notificaciones, conversaciones internas (chat) y la configuración de la página de inicio. Su propósito es ofrecer un entorno web unificado, moderno y adaptable para los usuarios del ERP, facilitando la interacción y personalización de la experiencia.
